Mobile mapping system

Developed embedded firmware using FreeRTOS with more than 30 concurrent real-time tasks executing under microsecond-level timing constraints. Integrated multiple sensors including accelerometers, gyroscopes, magnetometers, barometers, and dual GNSS modules to achieve accurate positioning and orientation estimation for aerial mapping applications.

Implemented high-speed interrupt-based data acquisition and synchronized sensor logging to SD card storage, alongside external flash memory management for system configuration and parameters. Designed and integrated communication interfaces including Bluetooth, Wi-Fi, and Ethernet (LAN) for real-time connectivity and data transfer. Developed precise multi-camera synchronization and triggering mechanisms supporting up to 6 simultaneous cameras for aerial photogrammetry and 2D mapping generation.

In parallel, developed Windows desktop applications using .NET Blazor, C#, JavaScript, HTML/CSS, and SQLite for real-time monitoring, visualization, processing, and analysis of sensor and navigation data. Also led hardware development activities including system architecture, hardware integration, debugging, and coordination of embedded hardware design tasks, with strong focus on real-time reliability, synchronization accuracy, and high-performance system operation.

2024 Medical · Firmware · Hardware · IEC 62304

Medical Auto Sampler system aligned with IEC 62304

Worked as Firmware Developer and Hardware Designer in the development of an automated medical autosampler system for laboratory blood sample processing. Designed the embedded hardware and developed firmware in Embedded C++ based on STM32F4 microcontrollers using FreeRTOS for reliable real-time operation. The system included a custom mechanical platform capable of holding up to 30 blood sample tubes and performing fully automated sample handling workflows.

Integrated a touchscreen user interface and USB HID keyboard connectivity, allowing operators to configure sample quantities and control the sampling process efficiently. Implemented RS232 communication with ProLYTE medical analysis devices, enabling automated command transmission, synchronization with sampling operations, and retrieval of analysis results directly from the laboratory equipment. Developed reliable data management functionality with storage capacity for up to 300 patient records and test results.

The project required robust real-time task management, hardware–software integration, and stable serial communication to ensure accurate and uninterrupted medical laboratory operation. Development practices were aligned with IEC 62304 expectations for medical device software lifecycle processes. The system was successfully manufactured and deployed in large quantities for laboratory and medical diagnostic environments.

2023 Aerospace · Firmware · Hardware

ARINC429 converter system

Worked as Firmware Developer and Hardware Designer in the complete development of a high-speed ARINC429 to Modbus TCP protocol converter for aerospace applications in Embedded C++ using STM32H7 microcontrollers. Designed the hardware architecture, schematics, and embedded firmware for a real-time communication system capable of interfacing directly with aircraft ARINC429 buses. The system supported 32 independent ARINC429 receive channels and 16 transmit channels, allowing simultaneous communication with multiple aircraft subsystems and engine control units.

Implemented high-performance bidirectional data conversion between ARINC429 and Modbus TCP over 100 Mbps Ethernet, enabling real-time data exchange between desktop software and aircraft engines. Developed the firmware using FreeRTOS with approximately 25 concurrent real-time tasks, with strong focus on deterministic execution, low-latency communication, and zero data loss operation.

Designed the system to handle continuous high-speed data streams reliably under strict real-time constraints, ensuring robust synchronization and stable operation in mission-critical aerospace environments.

2022 Automotive · Power Electronics · Firmware

On board converter system for Electric Vehicle

Worked as Embedded Software Developer and Hardware Designer in the development of an automotive-grade EV charger supporting single-phase (3.3 kW) and three-phase (10 kW) operation. Designed full power electronics hardware including schematic design, transformer/inductor calculations, control stage architecture, simulation, and 8-layer PCB layout with focus on EMC/EMI/ESD compliance (ISO 7637, ISO 16750) and functional safety considerations aligned with ISO 26262 principles.

Implemented embedded control firmware in Embedded C++ on TI C2000 microcontrollers using bare-metal architecture for deterministic real-time performance. Integrated CAN 2.0 communication (ISO 11898) for real-time vehicle–charger interaction and system monitoring.

Additionally developed a Qt Widgets desktop application for testing, debugging, and validation via serial-to-CAN interface. The project required end-to-end integration of embedded control, power electronics, and communication systems for high-reliability automotive EV charging systems.

2021 Navigation · Sensor Fusion · Firmware

Attitude and Heading Reference System (AHRS)

Worked as Firmware Developer and Hardware Designer in the development of a high-precision AHRS (Attitude and Heading Reference System) for real-time orientation and motion estimation applications. Designed the embedded hardware and developed firmware using Embedded C and Bare-Metal programming for sensor acquisition, signal processing, and real-time attitude estimation using gyroscope, accelerometer, and magnetometer sensor fusion.

Implemented advanced estimation and filtering algorithms including Kalman Filter–based approaches to accurately calculate Euler angles, quaternions, and angular velocity under dynamic operating conditions. Developed complex calibration and compensation algorithms for sensor alignment, bias correction, and magnetic calibration to improve long-term stability and measurement accuracy. Designed and integrated desktop software for real-time data visualization, logging, monitoring, and execution of calibration procedures through direct hardware communication.

Achieved high-accuracy orientation estimation with roll and pitch accuracy reaching 0.1° and yaw (heading) accuracy of 0.5°. The project required strong expertise in embedded systems, mathematical modeling, control systems, real-time signal processing, and hardware–software integration.
